IP 는 중복 이 될수 없다. 고유한번호
가짜 IP 라고 부르지 않는다. 라우터로 부터 부여받은 IP 를 (Private IP , 내부 IP)
진짜 IP 라고 부르지 않는다. 회선 에서 라우터 IP 를 (Public IP , 외부 IP)

다른 그룹의 네트워크에 접속하기 위해 외부 IP 를 알아야한다. 
공유기를 거쳐 외부 IP 로 접속 을 한다. (역순)

private IP 사용하는 이유 : 
보안성, IP주소 부족현상 해소,
특징 : 브로드 캐스팅 (네트워크에 연결된 모든 호스트에 전송되는 방식)

추적 : 외부 IP 까지 밖에 모르기 때문에 사용자 내에서 추적. 

포트 를 사용하는이유: 서비스가 많기 때문에 어떤 서비스인지 식별하기 위함.
결론 : 포트번호 알아야 함.

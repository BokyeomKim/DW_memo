상속

클래스(객체) 에 대한 개념이 부족하면 
상속이 힘들다...

자바 

프로젝트 ->패키지->클래스
(패키지를 만들지 않았어도 프로젝트를 만들었다면 
디폴트 패키지 생성.)
자바 버전 12 사용.(현 사용중 )
HTML 5 
JAVASCRIPT 5


1. 프로젝트 생성
	:프로젝트 이름은 대문자 or 소문자
	main_file(정답x)
2. 패키지 생성(생략 가능)
	:패키지 이름은 소문자(camel 표기법으로)
	mainFile  
3. 클래스 파일 생성
앞글자 대문자로 작성하도록 한다.


*public = public class가 두개가 올수 없다 . 
이름을 의미 한다.



new 를 이용해서 클래스를 호출할 때
생성자가 자동으로 호출됨 . 
호출과 동시에 초기화
